Introduction

Want to make a form for contact or sign up you need input.It is one the most important component for the user to interact with your web app.

Basic Input

Input is very simple and self validatory, that means it will react how you will react to it.Try putting en email in the input below and then try entering a random no. and you will see the difference.

Variations

By default the border given to the input is only the bottom one, if you want the full one you can use input--border and can further use 'br-*' for giving it border-radius
