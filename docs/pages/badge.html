Badge

Introduction

Badge are used for communicating a very subtle amout and type of information for e.g - products in carts, notifications on the user avatar etc. The structure of using it is very minimum so that you can use it with almost evry component out there.

Basic Badge

You can use it with literally any component, like text, headings, or icons below.

Badge Component

With Icon

By default FIT UI comes with fontawesome icons @5.15, which is used in the badge component, simpley give 'icon-true' and 'iconValue = ' to you desired fontawesome icon class and thats it. you are done.

How To...

there are bunch other values like radius , color and position which are there to modified.
Color can take three value 'pri', 'sec', 'def' for primary color , secondary color, and black.
Radius can take values like 'br-small', 'br-mid', 'br-large', 'br-full' for border radius of the badge.
Position can take values like 't-left', 't-right', 'b-left', 'b-right' for top-left , top-right, bottom-left and bottom-right.
