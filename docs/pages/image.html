Image

Introduction

Images are a very fundamental part of showing graphics to the user for interaction or make the content more engaging but styling making them responsive is very hard.

Basic Image

The images can be styled in a number of ways, cover or contain.by default it will have object-fit:contain. You can add 'image--cover' for object-fit:cover or 'image--file' for object-fit:fill;

Rounded Images

Using simple 'br-*' class you can add border-radius to the image
