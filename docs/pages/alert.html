Alert

Introduction

Alert makes easier for user to be aware of the results of their respective actions.It pop ups out of nowhere therefore it work even better for awareness

Basic Alert

Very simple alert componens which comes with Javascript. By default they will take a maximum of 45% width of the page, which makes them perfect for responsivenes, also the text which overlaps is by default hidded for UX.
