Slider

Introduction

Slider can be used on a various ways like product page, hero section on homepage or even as a simple sidebar card.

Basic Use

Just like every other components use the html code below and thats it.The default one comes with arrow buttons for changing the slide.

Auto SlideShow

Making the slideshow auto is very easy with the library just add an attribute of 'slideShow' and add the interval in ms and done.

Slide Show With Dots

Don't like side buttons, want something more modern dont worry we got you covered.Simply add a class of 'slider--dots' and done now instead of side buttons you would be using dots for slideshow
Note - As of now you can only apply one tpe of buttons wether side buttons or dots not both YET.
